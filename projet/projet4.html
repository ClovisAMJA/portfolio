C embarqué - Création d'un terminal SMS

  • Accueil

Présentation du projet

Ce projet portait sur la création d'un terminal SMS en langage C. L'objectif était de planifier le développement, de structurer le code et de gérer l'envoi d'un message texte.

Le fonctionnement reposait sur un codage binaire et une impulsion lumineuse, avec un travail réalisé dans STM32CubeIDE et VS Code.

Technologies utilisées

  • Langage C
  • STM32CubeIDE
  • VS Code
  • Codage binaire

Objectifs

  • Développer une solution embarquée
  • Mettre en œuvre un envoi de message
  • Organiser les étapes du projet
