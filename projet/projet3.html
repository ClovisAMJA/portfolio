SAE1.04 - Création d'une base de données

Présentation du projet

Ce projet consistait à construire une base de données relationnelle à partir d'un besoin client. J'ai travaillé sur le recueil des besoins, la modélisation et la structuration des données.

Le travail s'est appuyé sur UML, le modèle relationnel et des requêtes LMD pour organiser et exploiter les données de manière cohérente.

Technologies utilisées

  • UML
  • Modèle relationnel
  • Requêtes LMD
  • Visual Paradigm

Objectifs

  • Analyser le besoin du client
  • Concevoir une base cohérente
  • Rédiger les requêtes nécessaires
