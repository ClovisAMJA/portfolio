Présentation du projet

Ce projet consiste à créer un site de tourisme sur l'Espagne à partir d'un besoin client. J'ai travaillé sur la structure des pages, l'intégration HTML/CSS et la lisibilité générale.

Le but était de produire un site clair, cohérent et agréable à consulter, tout en utilisant GitHub pour organiser le travail.

Technologies utilisées

  • HTML / CSS
  • GitHub
  • Analyse du besoin client

Objectifs

  • Structurer un site web à partir d'un besoin réel
  • Créer une interface lisible et cohérente
  • Garder une organisation propre du projet
